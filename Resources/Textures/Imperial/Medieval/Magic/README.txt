Убедительная просьба придерживаться следующего формата при размещении спрайтов для заклинаний:

1. Создать папку (не .rsi) с названием, равным названию папки с прототипом заклинания

2. Внутри папки создать:
   icons.rsi, где будут расположены иконки для action и store.
     примечание: если иконка только одна, то название всё равно icons.rsi, а не icon.rsi
   projectiles.rsi, где будут расположен спрайт самого заклинания (если есть)
     примечание: аналогично примечанию выше

3. Названия .png файлов и state'ов в meta.json должны быть следующего формата: 
   beginner, middle, senior в зависимости от уровня. 
   При необходимости добавить приписку (Например beginner_aim)
   В заклинании MagicArrow пример того, как офоромляются заклинания, при разветвлениях (спрайтов иконок больше 3)
   Просьба не называть их "base", "1", "fireball" и т.д. 

4. Если для заклинания используется уже существующий спрайт (например projectile для fireball взят из спрайтов визардов), 
   то создавать папки и копировать туда этот спрайт не нужно
